# Roundtable product roadmap

## Now — usable human/agent collaboration

- Independent provider selection, including Codex × Codex.
- Browser-based run creation and live conversation visualization.
- Human gates, interventions, pause, resume, and cancellation.
- Persistent Project Room context for long-running projects.
- Linked discuss → plan → human-confirmed build workflow with explicit owners and scope.
- Tamper-evident external audit mirrors, exact prompt capture, heartbeats, and owner leases.
- Hardened local web boundary and stable live controls.

## Next — durable project execution

- Persist collaboration state so interrupted and server-restarted runs can resume.
- Replace duplicated run files with a typed append-only event log and generated projections.
- Expand linked plan batches into a project-wide task graph with milestone rollups.
- Branch a run from any message and compare alternative decisions side by side.
- Show code changes, test results, artifacts, and blocking issues as first-class cards.
- Add per-run token, duration, and estimated cost budgets with stop conditions.
- Integrate git worktrees and explicit human approval before commits or destructive actions.

## Later — a multi-agent project room

- Provider plugins for local models and additional coding-agent CLIs.
- Protocols beyond leader/reviewer: expert panel, debate, security review, red team, and majority/consensus voting.
- Route work by capabilities, availability, cost, and context-window health.
- Search and summarize all project decisions, disagreements, evidence, and outcomes.
- Export/import complete project rooms and generate handoffs for another person or agent.
- Optional authenticated LAN/remote access with role-based permissions.
- Notifications when human input is required, including desktop and chat integrations.
- Evaluation dashboards that compare providers and protocols on real project outcomes.
